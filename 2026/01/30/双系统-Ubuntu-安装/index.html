双系统 - Ubuntu 安装

发表于 2026-01-30 分类于 工作环境配置
已有 Windows 系统，安装 Ubuntu 系统。

安装双系统

参考链接：

  1. (保姆式教学) Win10 + Ubuntu 20.04——双系统安装方法 + 配置显卡 + root权限 + flash调配_保姆式教学 win10+ubuntilu-CSDN博客
  2. Windows10怎么使用Rufus制作USB启动盘？ - 知乎 (zhihu.com)
  3. Ubuntu18.04安装教程_ubuntu 18.04-CSDN博客

  • 准备工作：

    • 未安装 Ubuntu 系统的电脑一台，16GB 的 U 盘一个

  • 实际操作：

    1. 内存管理：

      Win + X 打开磁盘管理

      • 需要预留大概 200 GB 的未分配空间——右键压缩卷（具体操作请参见参考链接1）
      • 需要知道各磁盘对应的名字——在Ubuntu正式安装时用于分配内存

    2. 下载 iso（光盘映像）文件：

      下载地址：Get Ubuntu | Download | Ubuntu，大概需要半个小时（不需要梯子）

    3. 利用Rufus将U盘制作为启动盘：

      • 下载rufus软件：Rufus - 轻松创建 USB 启动盘
      • 启动rufus并选择2中下载的iso文件，按照参考链接2进行操作

      注意：需要在进行操作前确定自己电脑的 bios 类型

      • 参见用 Rufus 制作 Ubuntu 系统启动盘时,选择分区类型为MBR还是GPT？_rufus分区类型怎么选择-CSDN博客

      • 查看bios模式：

        
graph LR
Win+R --> cmd
cmd --> msinfo32

    4. 使用U盘，安装新系统：

      • 在插入U盘的前提下重启，在开机的时候狂按F2进入bios界面，并进行如下操作：

        • 将Secure Boot改为disabled

        • 在启动模式中选择非Windows启动的方式，并选择允许USB启动

          如bios模式是UEFI，则选择与其不同的那个。反正具体自己多试试，总能到安装那一步的

    5. 正式安装：

      • 分配内存：按照参考链接1or3进行
      • 启动，正常安装即可
