pip & conda 教程及其区别

发表于 2026-01-30 分类于 Python
简述 pip 与 conda 的使用方法及其区别。

在虚拟环境中安装package

使用pip安装：

  • pip install <package>，如果不知道package的具体名称，可以查阅PyPI · The Python Package Index

  • 加速安装命令：

    pip install <package_name> -i https://pypi.tuna.tsinghua.edu.cn/simple

    （在关闭/开启加速器的情况下，均可使用）

使用conda安装：

  • conda install <package>

conda的安装会指定channel，与channel相关的指令有

  • 查看conda的下载渠道：conda config --show channels或conda config --get channels

  • 添加渠道：
    conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

  • 移除渠道：
    conda config --remove channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

pip与conda的区别：

Python - Conda - 对比 conda 和 pip - frank_cui - 博客园 (cnblogs.com)

下载后channel的区别

在虚拟环境中，输入命令conda list可以看到最后一列是Channel，其中

  • pip安装的库对应的channel是pypi-Python Package Index：PyPI
  • conda安装的库对应的channel多种多样，包括但不限于defaults, conda-forge, nvidia, pytorch。
    • 在此，笔者建议，在使用conda安装相关库的时候，最好加上-c <channel_name>
