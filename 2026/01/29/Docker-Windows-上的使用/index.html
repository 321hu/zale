在 Windows 上使用 Docker

新手入门Docker之Windows下如何使用Docker_windows docker 使用-CSDN博客

  1. 运行已下载好的镜像：
  • docker images：查看所有已拉取镜像
  • docker run --name <container_name> -d <image>:<tag>：在后台运行镜像，且将此容器命名为container_name
  1. 运行具体容器
  • docker ps -a：查看所有容器
  • docker start ID 或 docker start <container_name>：启动具体容器
  • docker exec -it [容器ID或名称] /bin/bash：进入容器终端
    • exit 退出容器终端
    • docker stop [容器ID或名称]：关闭容器
  1. 删除某个容器
  • docker rm [容器ID或名称]
  1. 在创建容器时，使容器某路径与 Windows 文件夹挂载
  • docker run -it --name <container_name> -v d/LibSoftware/:/root/MyFolder <image>:<tag>
    • -it 命令使得创建之后直接在终端打开
    • --name <> 可以自定义容器名称
    • -v <Windows_path>:<Docker_path> 挂载
