安装插件

  • Markdown All in One：
    几乎所有Markdown用得上的功能在 Markdown All in One 里面都有：自动补全、自动生成 Table of contents、Math、……
  • Markdown Preview Enhanced：
    Markdown Preview Enhanced 提供了比 Markdown All In One 更为丰富好用的预览功能，包括调整预览视图下的主题、通过 chrome 导出 pdf、png、html 文件等功能。
